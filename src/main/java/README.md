# Requirements

--------------

JDK 19 with Loom Preview: [https://jdk.java.net/loom/](https://jdk.java.net/loom/)

Compile with arguments: `--release 19 --enable-preview`

Run with VM arguments: `--enable-preview`

Example:

    /path/to/loom/jdk/bin/java --enable-preview duel.threads.Main 1 virt 1000000
